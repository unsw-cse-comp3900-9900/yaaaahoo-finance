### Node.js required: https://nodejs.org/en/download/
### ```npm i```

### ```npm start```

#### Make sure to fill config.js file
``` 
export const config = {
    "newsApiToken": "",
    "alphaVantageApiToken" : "",
    "worldTradingApiToken" : "",
    "iexCloudApiToken" : "",
    "apiKey": "",
    "authDomain": "",
    "databaseURL": "",
    "projectId": "",
    "storageBucket": "",
    "messagingSenderId": "",
    "appId": "",
    "measurementId": ""
};
```

Will run on http://localhost:3000/
